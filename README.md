# Balanza

A Java application for managing financial information using SQLite database

## How to Run

1. Install Java 17
2. Clone this repo
3. Run `Main.java` from IntelliJ

## Dependencies

- Java 23
- SQLite3 JDBC Driver
- IntelliJ IDEA
